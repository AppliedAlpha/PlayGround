### React Feature
- ```Component```: Consists UI
- ```Virtual DOM```: 바뀐 데이터로 그리고 비교함 → 바뀐 부분만 Re-Render (DOM 변화 최소화)
- ```JSX (JavaScript XML)```: HTML 문법과 유사하게 작성 가능
 ```jsx
 const element = <h1>Hello, React</h1>; // JSX Ex.
 ```
 - ```Status Management, Routing, API Communication```
